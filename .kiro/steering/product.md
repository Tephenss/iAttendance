# Product Overview

**iAttendance** is an Android mobile application for attendance management in educational institutions.

## Core Features
- Multi-role authentication system (Student, Teacher, Admin)
- Firebase-based user authentication and data storage
- Email verification system
- Role-based dashboard interfaces
- Secure password handling with BCrypt encryption

## Target Platform
- Android mobile application
- Minimum SDK: API 24 (Android 7.0)
- Target SDK: API 36 (Android 14+)

## Key User Roles
- **Students**: View and manage their attendance records
- **Teachers**: Mark attendance and manage class records
- **Administrators**: Oversee system-wide attendance data and user management
